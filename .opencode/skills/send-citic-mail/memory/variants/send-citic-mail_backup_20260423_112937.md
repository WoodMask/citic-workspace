# Skill: send-citic-mail

发送邮件到指定公司邮箱。支持发送附件文件和正文内容。

## 触发条件

当用户说以下内容时触发此技能：
- "发送**附件给我公司邮箱"
- "发送**内容给我公司邮箱"
- "发邮件到公司邮箱"
- "发到我邮箱"
- "把**发到我邮箱"
- "发送**到公司邮箱"

## 配置信息

配置文件位置：`config.json`（相对于skill目录，与send_email.py同级）

配置项说明：
| 字段 | 说明 | 默认值 |
|------|------|--------|
| sender | 发件人邮箱 | maskmagician@qq.com |
| receiver | 收件人邮箱 | wusinan_kzx@citicbank.com |
| auth_code | SMTP授权码 | 需自行获取 |
| smtp_server | SMTP服务器 | smtp.qq.com |
| smtp_port | SMTP端口 | 465 |
| default_subject | 默认邮件标题 | 工作文件发送 |

**迁移到其他机器时**：只需修改 `config.json` 中的配置即可，无需修改代码。

## 使用方式

### 发送附件

用户说：`发送报告.docx给我公司邮箱`
用户说：`把当前目录下的评估报告发到我邮箱`

Agent应：
1. 解析用户指定的文件路径
2. 如果是相对路径，转换为绝对路径
3. 支持通配符匹配（如 `*.docx`）
4. 调用脚本发送

### 发送内容

用户说：`发送以下内容给我公司邮箱：今天是周五`
用户说：`发邮件到公司邮箱，内容是项目已完成`

Agent应：
1. 提取用户指定的正文内容
2. 调用脚本发送

### 发送附件+内容

用户说：`发送报告.docx给我公司邮箱，内容是请查收报告`

Agent应：
1. 提取附件路径
2. 提取正文内容
3. 调用脚本发送

## 脚本调用

脚本位于skill目录下的 `send_email.py`，调用方式：

```bash
python <skill_dir>/send_email.py \
  --attachments "文件路径1" "文件路径2" \
  --content "邮件正文内容" \
  --subject "邮件标题"
```

其中 `<skill_dir>` 为本skill所在目录。Agent应根据当前项目结构定位skill目录。

调用示例（假设skill在项目根目录下的 .opencode/skills/send-citic-mail/）：
```bash
python .opencode/skills/send-citic-mail/send_email.py -a "report.docx" -c "请查收报告" -s "工作报告"
```

### 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| --attachments | 否 | 附件文件路径，可多个 |
| --content | 否 | 邮件正文内容 |
| --subject | 否 | 邮件标题，默认"工作文件发送" |

## 默认邮件正文

如果用户未指定正文内容，使用以下模板：

```
您好！

附件为工作文件，请查收。

发送时间：{YYYY-MM-DD HH:mm:ss}

如有问题，请随时联系。
```

## 执行流程

1. **解析用户请求**：识别附件路径、正文内容、邮件标题
2. **验证文件存在**：检查附件文件是否存在
3. **构建邮件内容**：使用用户指定正文或默认模板
4. **调用发送脚本**：执行 send_email.py
5. **返回结果**：告知用户发送成功或失败原因

## 注意事项

1. 附件文件路径支持绝对路径和相对路径
2. 支持通配符匹配多个文件（如 `*.pdf`）
3. 正文内容支持多行文本
4. 发送前确认附件文件存在
5. 大附件（>20MB）需提示用户可能发送失败
6. QQ邮箱单邮件附件总大小限制50MB
