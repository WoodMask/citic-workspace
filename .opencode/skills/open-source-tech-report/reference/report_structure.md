#### 常见开源协议说明

| 协议 | 商业友好度 | 主要要求 |
|------|------------|----------|
| Apache-2.0 | 高 | 保留版权声明，修改需说明 |

| GPL | 低 | 必须开源，不可闭源商业使用 |
| LGPL | 中 | 动态链接可闭源，修改库需开源 |
| BSD | 高 | 保留版权声明 |

### 1.8 供应链风险评估

**风险评估维度**：

| 维度 | 低风险 | 中风险 | 高风险 |
|------|--------|--------|--------|
| 开发者稳定性 | 头部公司/知名团队 | 中型团队 | 个人开发者 |
| 社区活跃度 | Star>10000,活跃更新 | Star>1000,定期更新 | Star<1000,不活跃 |
| 协议合规性 | Apache/MIT等友好协议 | LGPL等有限制 | GPL等强制开源 |
| 公司资质 | 已备案/合规运营 | 合规待完善 | 无备案/违规风险 |

**评估结论格式**：
```

（3）{协议合规性说明}
（4）{其他因素}
```
